Opencart Language Editor
=========================================

Features
---------------
- Edit language files without overwriting core language files. ALl changes are saved in file with a underscore at the end.
- Search for text you need to change
- Easily modify text and use HTML editor 


Install
==========================================
1) Use the Extension Installer selecting: upload.ocmod.zip
Or upload the files manually and copy /install.ocmod.xml to: /system/language_editor.ocmod.xml

2) Go to modifications and click Refresh icon. 


Support
